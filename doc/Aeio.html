val async : ?ctxt:context -> ('a -> 'b) -> 'a -> 'b promise
async f v spawns a fiber to run f v asynchronously. If no cancellation context was provided, then the new fiber shares the cancelallation context of the caller.
val await : 'a promise -> 'a
Block until the result of a promise is available. Raises exception e if the promise raises e.
Raises Promise_cancelled if the promise was cancelled.
